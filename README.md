Voice-Enabled Digital Assistant for Senior Citizens

Sahaaya is a voice-enabled digital assistant designed to help senior citizens easily access essential services and reminders through natural language interaction.

Instead of navigating complex smartphone interfaces, users can speak commands to the assistant, which interprets the request and performs tasks such as setting reminders, contacting emergency services, or retrieving useful information.

The project focuses on accessibility, simplicity, and voice-driven interaction to make technology more usable for elderly users.
